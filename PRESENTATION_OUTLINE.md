## Slide 6: Game Flow
**Title:** How the Game Works

**Content:**
1. **Start Game** → Generate password with cycling strength (weak → medium → strong)
2. **Brute Force Attack** → Automated sequential attempts + manual guessing option
3. **Security Analysis** → Entropy, crack time, vulnerability assessment
4. **Decryption Game** → Pattern-based key cracking
5. **Final Summary** → Complete vulnerability report
6. **Gift Screen** → Reward for beating brute force
7. **Leaderboard** → Save and compare scores
