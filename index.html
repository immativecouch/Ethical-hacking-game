üéÆ How to Play

  1. Select a target device (Laptop, Mobile, or PC)
  2. Choose difficulty level (Easy, Medium, or Hard)
  3. Use arrow keys or WASD to move the selection box
  4. Find the correct password by selecting characters
  5. Complete the file retrieval sensor operation
  6. Decrypt files if playing Medium or Hard difficulty

üîí Cybersecurity Education

This game demonstrates common attack vectors:

  • Brute Force Attacks: Attackers try multiple password combinations
  • Social Engineering: Gathering information to guess passwords
  • Data Interception: Retrieving files from compromised devices
  • Encryption Bypass: Attempting to decrypt protected data

üõ°Ô∏è How to Protect Yourself

  • Use strong, unique passwords (12+ characters, mixed case, numbers, symbols)
  • Enable two-factor authentication (2FA)
  • Keep software and systems updated
  • Use encryption for sensitive files
  • Be cautious of phishing attempts
  • Monitor your accounts for suspicious activity
  • Use a password manager
  • Never share passwords or personal information

ABOUT

Ethical Hacker is an educational cybersecurity training game designed to demonstrate common attack vectors and security vulnerabilities in a safe, controlled environment.

Project Purpose

This project aims to educate users about:

  • How brute force attacks work
  • Common security vulnerabilities
  • Best practices for protecting digital assets
  • The importance of strong passwords and encryption

Educational Value

By experiencing these attack scenarios in a game format, users can better understand the importance of cybersecurity and learn how to protect themselves from real-world threats.
